# The Book Haven

## Description
The Book Haven is a book search website that allows users to search for books using the Google Books API and Open Library API. It provides accurate search results, book details, and recommendations based on the searched book, author, and series.

## Features
- Search for books using titles and author names.
- Displays detailed book information.
- Provides recommendations for similar books, books from the same author, and books from the same series.
- Search caching for improved performance while allowing new searches.
- Search triggered by pressing the Enter key.
- Enhanced search accuracy with keyword-based search and prioritized relevance.
- User authentication and data storage using Firebase.

## Technologies Used
- HTML, CSS, JavaScript
- Google Books API
- Open Library API
- Firebase (Authentication and Database)

## Installation Instructions
1. Download or clone the repository.
2. Extract the files if downloaded as a ZIP.
3. Set up Firebase by adding your Firebase configuration to the project.
4. Open the `index.html` file in a web browser.

## Usage
1. Enter a book title or author name in the search bar.
2. Press Enter to initiate the search.
3. Browse the search results and click on a book for detailed information.
4. View recommendations related to the selected book.
5. Sign up or log in to save searches or bookmarks (if applicable).
